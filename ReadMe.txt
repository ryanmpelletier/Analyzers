========================================================================
    CONSOLE APPLICATION : Analyzers Project Overview
========================================================================
This is a C++ project for my Theory of Programming Languages class. 

A lexical analyzer analyzes text and classifies tokens. 

For example, in most languages:

i = 5;

May be interpreted as IDENTIFIER EQUALS INT_CONSTANT SEMICOLON.

This syntactical analyzer will correctly identify that the given input file
satisfies a BNF language grammer.
